# Ruboty::Todo

ruboty handler to manage your todo list

## Usage

### Setup

set environment variables `ENV['TODO_OWNERS']` to check message from owners

- join owner's name by comma like below:
  - `owner_alice,owner_bob,owner_charlie`
- ignore to check
  - `IGNORE_CHECK`

### Show Help

`ruboty todo help`
